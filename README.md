# 📊 Power BI Financial Sales Dashboard

A dynamic and interactive Power BI dashboard built from a financial dataset to analyze:

- 📈 Monthly sales & profit trends
- 🌍 Regional performance (country-wise)
- 🛍️ Top-selling products & product profitability
- 🧠 KPI cards and drill-through navigation

---

## 🔧 Tools & Features

- Power BI Desktop
- DAX Measures (Profit %, Total Sales, Top KPIs)
- Slicers & filters for deep dive analysis
- Custom visualizations: Donut, line chart, bar chart
- Drill-through and tooltip pages
- Clean and executive-level layout

---

- `Financial_Sales_Dashboard.pbix`: Full Power BI project
- `Dashboard_Summary.pdf`: Static snapshot version
